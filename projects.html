A&M Bread Express

Github: here!
Fully functional mock retail bread retail store. The site was a product of a
semester's worth of cumulative project work, beginning with the ERD design,
transitioning into the model and backend setup, and topping it off with the front end.
Fantastic experience with full stack development using Ruby on Rails, creating
a robust web application.

Features include:
  • -Account creation, different account types, user authentication and authorization

  • -Viewable and modifiable customer, address, product information (restricted to certain users)

  • -Adding/removing from cart, checking out order, credit card authentication

  • -Responsive design with Materialize framework

Defender

A side-scrolling space shooter written in Python with the Pygame module.
Done over the course of three weeks as a class term project, it was a great
exercise in pacing and object oriented design. Plus tons of fun to modify sprites
and pay homage to all the similar games I've played over the years!
